# Virtual--Christmas--Card
Este cartão virtual foi criado com foco em **experiência, emoção e simplicidade**. A proposta é ir além de uma simples página, oferecer uma simles jornada visual: um gesto simbólico de abrir um envelope para revelar uma mensagem especial. O projeto também demonstra boas práticas de organização, separação de responsabilidades e atenção à UX.
